## Instalar

### Dependencias:
```
node 18+
```

## Instalando e rodando o projeto

Clone este projeto em algum diretório da sua escolha. Em seguida, instale as dependências usando npm.
Para iniciar, você pode rodar usando npm run dev ou gerar uma build com npm run build e após terminar, npm run start

```bash
git clone https://github.com/motadv/psi-facil-prototipo.git

cd psi-facil-prototipo/

npm install

npm run dev
```

Após inicializado, o projeto estará online em localhost:3000 no seu navegador de preferência.

## Fechando o projeto

Quando quiser encerrar a execução, apenas interrompra a execução do program com CTRL+C ou fechando o terminal que está executando o projeto.
